== Description ==

C4D Shoppable Images increases traffic through your website by creating shoppable images: images containing different points which link to various products.

### [Demo 1](http://30tet.coffee4dev.com/lookbook-images/ "live demo") [Demo 2](http://30tet.coffee4dev.com/shoppable-image-slider-style/) [Demo 3](http://30tet.coffee4dev.com/2018/07/09/duis-imperdiet-felis-tortor-non-malesuada-diam-iaculis-vitae/) [Document](http://coffee4dev.com/docs/document-c4d-woo-shoppable-images/ "document") [Premium](http://coffee4dev.com/woocommerce-shoppable-images/ "pro version") [Support](http://coffee4dev.com/contact/ "Support & Contact")

#### FEATURES IN C4D Shoppable Images
- Completely integrate with WooCommerce plugin
- Using shortcode in page
- Using shortcode in post
- Using shortcode in widget 
- Using shortcode in pagebuilder Elementor, Bavebuilder, Visual Composer.
- Display with slider style
- Options for slider style
- Display with grid style (4 columns, 3 columns, 2 columns, 1 column)
- Option for grid 
- Drag & drop admin to create collection.
- Support category, tags, single page.
- Duplicate collection
- Custom option to modify style of product title
- Custom option to modify style of click icon
- Custom option to modify style of product desc
- Custom option to modify style of product images
- Custom option to modify style of product button

### INSTALLATION & CONFIG

1. Upload `c4d-shoppable-images` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

### OPTION PAGE

To enable admin options please install [C4D Plugin Manager plugin](https://wordpress.org/plugins/c4d-plugin-manager/)

== installation ==

1. Upload `c4d-shoppable-images` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create collection in Shoppable Images -> Add New 
4. Insert shortcode in page or post, widget.
